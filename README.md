### SETUP

#### 1.Install XAMPP or WAMPP

#### 2.Open XAMPP Control panal and start [apache] and [mysql] .

#### 3. Clone the project from https://github.com/monyslim/lister.com.git into /opt/lampp/htdocs

#### 4. Build an image and run it to see if it's working

#### 5. Stop & rm the running container

#### 6. Write your docker-compose.yaml file and run it. Use mysql:5.7 for the mysql container image

#### 7. Exec into your application conatiner and cd into "includes" change the necessary things you need to change

#### 8. Exec into your mysql container and do the necessary things you need to do

#### 9. Go to your browser and type: localhost:91/tables.php to generate the database and localhost:91/login.php to sign up

#### 10. Sign up. After siginig up, sign in with the details you provided when signing up

#### 11. Create a simple post, make it public
